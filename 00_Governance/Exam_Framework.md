# Exam Framework (Regulation-Aligned)

기준 문서: `00_Governance/Qualification_Management_and_Operation_Regulations.md`

## 1. Basic
- 검정방법: 필기검정
- 검정과목:
  - 분산 신뢰 구조의 이해
  - WEB3 기초 구조의 이해 (지갑/주소/트랜잭션/컨트랙트/토큰 기본 개념)

## 2. Associate
- 검정방법:
  - 필기검정: WEB3 아키텍처 분석
  - 실기검정: 권한 통제 설계 / 자금 흐름 기반 리스크 평가

## 3. Professional
- 검정방법:
  - 실기검정: 프로토콜 설계 / 거버넌스 설계 / 컴플라이언스/운영 설계

## 4. 평가 설계 원칙
- 실제 자산 이동을 요구하지 않는다.
- 시나리오 기반 과제로 절차 이해와 구조적 판단 능력을 검정한다.
- 합격기준의 세부 점수/비율은 별도 운영 공고 문서에서 관리한다.
