# ch4. 분산 신뢰 3층 구조 - Writing Expansion

## (집필 확장용) 이 강의를 책으로 옮기면 이렇게 된다
- 1. 신뢰를 분해하라: 합의·실행·검증  
- 2. 실행: 상태 전이 규칙이 세계를 만든다  
- 3. 검증: 믿지 않고 확인하는 구조  
- 4. 합의: 하나의 역사를 선택하는 규칙  
- 5. 트랜잭션 라이프사이클 8단계(도식)  
- 6. 레이어별 장애/사고 진단 프레임  
- 7. 실습: 익스플로러로 3층 흔적 읽기  
- 8. 다음 장 예고: 키 기반 소유권 구조(서명은 무엇을 증명하는가)
