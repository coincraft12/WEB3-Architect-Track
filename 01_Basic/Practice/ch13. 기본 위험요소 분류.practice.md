# ch13. 기본 위험요소 분류 - Practice

## 미니 실습(25~30분) — 위험요소 체크 카드 만들기
### 실습 주제
12강에서 만든 구조 해석 카드 1장을 가져와,  
위험요소 9종 체크를 붙여 “위험요소 카드”로 완성한다.

실습 체크포인트:
1) approve가 있는가? 있다면 범위(한도/기간)는?  
2) 서명은 트랜잭션 서명인가, 메시지 서명인가?  
3) 컨트랙트 운영 권한(owner/role)은 있는가? (있다면 무엇을 할 수 있나?)  
4) 외부 의존(프론트/서버/오라클)은 있는가?  
5) 조건 파라미터(슬리피지/데드라인)가 사용자의 손해를 허용하는가?

**출력(산출물)**: “위험요소 카드 1장”
- 체크된 위험요소(R1~R9)
- 각 위험요소별 피해 형태 1문장
- 예방 행동 1개씩
