# ch10. 가스(Gas) 조건 정의 - Practice

## 미니 실습(20~25분) — “가스 관점으로 트랜잭션 하나 읽기”
### 실습 주제: 트랜잭션 2개 비교 (성공 vs 실패/지연)
가능하면 같은 종류의 행동(예: 단순 전송 2개, 혹은 스왑 2개)을 고른다.

- gas limit / gas used  
- 수수료(총 비용)  
- 포함까지 걸린 시간(또는 confirmations)  
- 결과(success/revert/pending)  
- 가스 관련 추정 원인(혼잡? 한도? 로직?)

**출력(산출물)**: “가스 읽기 카드 2장” + 비교 결론 5문장
