# ch6.5. 블록체인 최소 정의 - Practice

## 미니 실습(10~15분) — ‘블록체인 4문장 정의’로 설명하기
### 실습 과제
블록체인을 누군가에게 설명한다고 가정하고, 아래 조건으로 5문장을 작성한다.

**조건**
- “코인”, “투자”, “채굴”, “PoW/PoS” 단어 금지  
- 상태/트랜잭션/합의/순서 고정은 반드시 포함  
- 마지막 문장은 “그래서 키/주소가 필요하다”로 연결

**출력 예시 형태**
- 문장1: 상태 정의  
- 문장2: 트랜잭션 정의  
- 문장3: 합의 정의  
- 문장4: 블록/체인 의미  
- 문장5: 키/주소 연결
