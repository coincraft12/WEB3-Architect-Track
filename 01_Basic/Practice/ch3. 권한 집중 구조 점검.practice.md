# ch3. 권한 집중 구조 점검 - Practice

## 미니 실습(15분) — 내 서비스 권한 지도 그리기
### 실습 주제: “내가 쓰는 서비스 1개를 권한-책임 관점으로 해부”

- 승인자(누가 OK를 내리는가?)  
- 거부자(누가 NO를 내릴 수 있는가?)  
- 되돌리는 자(누가 취소/환불/정정할 수 있는가?)  
- 규칙을 바꾸는 자(누가 수수료/정책을 바꾸는가?)  
- 피해 비용 부담자(문제 생기면 누가 손해를 떠안는가?)

**출력(산출물)**: 권한 지도 1장 + “가장 위험한 불일치 1개” 문장
