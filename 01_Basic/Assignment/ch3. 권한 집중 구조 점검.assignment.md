# ch3. 권한 집중 구조 점검 - Assignment

## 과제
### 과제 1: 권한-책임 불일치 리포트(1페이지)
- 서비스 2개를 고른다.  
- 각 서비스에서 동결/차단/규칙변경 중 2개 패턴을 찾아  
  - “누가 할 수 있는가(권한)”  
  - “누가 책임지는가(책임)”  
  를 비교한다.
- 마지막 문장: “이 서비스의 가장 큰 구조적 리스크는 무엇인가?”

### 과제 2(선택): 완화 설계 제안 3개
- 권한 분산 관점 1개  
- 책임 강화 관점 1개  
- 투명성/절차 강화 관점 1개  
으로 완화책을 제안한다.
