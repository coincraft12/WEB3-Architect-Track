# ch6. 트랜잭션 구조 정의 - Assignment

## 과제
### 과제 1: 트랜잭션 3종 분석 리포트(각 0.5페이지, 총 1.5페이지)
아래 3개를 각각 5요소(주체/대상/행동/조건/결과)로 해석한다.

1) EOA→EOA 송금 1건  
2) 컨트랙트 호출 1건(DEX 스왑/토큰 전송 등 아무거나)  
3) 실패(revert) 트랜잭션 1건

### 과제 2(선택): “기록” 관점 문장 고치기
아래 문장을 “행동/구조” 관점으로 바꿔쓰기
- “트랜잭션은 송금 기록이다.”  
- “가스는 수수료다.”  
- “실패한 트랜잭션은 아무 의미가 없다.”
