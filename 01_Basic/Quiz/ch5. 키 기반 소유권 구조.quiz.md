# ch5. 키 기반 소유권 구조 - Quiz

## 객관식 퀴즈 (4지선다)

### 문제 1
키 기반 소유권 모델의 핵심 문장으로 가장 적절한 것은?
A. 내 아이디=내 소유권
B. 내 키=내 통제권
C. 내 비밀번호=내 체인
D. 내 앱=내 자산
- 정답: B
- 해설: 개인키가 실제 통제권의 근거라는 점이 ch5 핵심이다.

### 문제 2
서명(Signature)이 증명하는 것으로 옳은 것은?
A. 사용자의 주민등록번호
B. 해당 행동에 대한 키 통제권 기반 승인
C. 네트워크 속도
D. 토큰 시세
- 정답: B
- 해설: 서명은 신원서류가 아니라 키 소유자의 의사 승인 증거다.

### 문제 3
중앙화 계정과 키 기반 구조의 차이로 맞는 것은?
A. 중앙화는 복구가 불가능하다
B. 키 기반은 기관이 항상 복구해준다
C. 키 기반은 사용자가 복구/보안 책임을 더 많이 진다
D. 중앙화는 서명을 사용하지 않는다
- 정답: C
- 해설: Basic 관점에서 가장 큰 차이는 책임 위치 이동이다.

### 문제 4
시드 구문(Seed Phrase)의 역할은?
A. 수수료 할인 코드
B. 키 재생성/복구를 위한 백업 정보
C. 합의 알고리즘 선택
D. 컨트랙트 배포 주소
- 정답: B
- 해설: 시드는 지갑 통제권 복구 수단이다.

### 문제 5
"주소를 안다"와 "자산을 통제한다"의 관계로 맞는 것은?
A. 완전히 동일하다
B. 주소를 알면 즉시 통제 가능하다
C. 주소는 좌표이고 통제는 개인키에 의해 결정된다
D. 주소는 오프체인 개념이다
- 정답: C
- 해설: 주소 문자열만으로는 통제권이 생기지 않는다.
