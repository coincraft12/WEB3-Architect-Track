# ch5. 키 기반 소유권 구조 - Quiz

## 퀴즈(10문항, 복습용)
1) 개인키(Private Key)는 무엇을 할 수 있게 해주는가?  
2) 공개키(Public Key)는 무엇을 위해 쓰이는가?  
3) 주소(Address)와 계정(Account)의 차이를 설명하라.  
4) 서명이 증명하는 것은 신원인가, 통제권인가?  
5) “키 기반 소유권”에서 소유권의 정의를 한 문장으로 써라.  
6) 중앙화 계정 모델이 제공하는 복구 가능성은 왜 가능한가?  
7) 키 기반 모델에서 복구가 어려운 구조적 이유는 무엇인가?  
8) 키 기반 소유권의 대표 리스크 5종을 쓰시오.  
9) 단일 키 구조가 위험한 이유를 ‘단일 장애점’ 관점에서 설명하라.  
10) “키 → 서명 → 검증 → 실행” 흐름을 한 문장으로 요약하라.
