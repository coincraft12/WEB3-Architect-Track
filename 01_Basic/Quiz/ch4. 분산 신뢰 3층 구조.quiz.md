# ch4. 분산 신뢰 3층 구조 - Quiz

## 객관식 퀴즈 (4지선다)

### 문제 1
분산 신뢰 3층으로 옳은 조합은?
A. 합의/실행/검증
B. 지갑/주소/토큰
C. 블록/채굴/브릿지
D. 승인/서명/가스
- 정답: A
- 해설: ch4는 신뢰를 합의-실행-검증으로 분해해 설명한다.

### 문제 2
실행(Execution)의 역할은?
A. 네트워크 이름 부여
B. 트랜잭션 규칙 처리로 상태 변경
C. 수험자 인증
D. 정책 공지
- 정답: B
- 해설: 실행은 상태 전이를 실제로 발생시키는 계층이다.

### 문제 3
검증(Validation)의 핵심은?
A. 어떤 기록을 공식 역사로 선택
B. 요청이 규칙을 만족하는지 확인
C. UI 문구 번역
D. 법적 분쟁 해결
- 정답: B
- 해설: 검증은 유효성 판단, 합의는 채택 결정이다.

### 문제 4
합의(Consensus)의 역할로 옳은 것은?
A. 개인키 생성
B. 블록 채택 순서를 하나로 맞춤
C. 토큰 승인 요청
D. 주소 라벨링
- 정답: B
- 해설: 합의는 네트워크가 단일 기록 순서를 공유하게 한다.

### 문제 5
"검증=합의"라는 문장이 틀린 이유는?
A. 둘 다 영어라서
B. 검증은 규칙 확인, 합의는 역사 채택으로 기능이 다르기 때문
C. 검증은 오프체인만 가능해서
D. 합의는 수수료가 없어서
- 정답: B
- 해설: 두 계층은 연동되지만 역할이 동일하지 않다.
