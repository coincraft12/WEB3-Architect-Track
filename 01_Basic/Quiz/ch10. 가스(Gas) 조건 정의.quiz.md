# ch10. 가스(Gas) 조건 정의 - Quiz

## 객관식 퀴즈 (4지선다)

### 문제 1
가스의 역할로 가장 적절한 것은?
A. 예쁜 영수증 출력
B. 실행 자원 계량/상한/우선순위 관리
C. 주소 길이 조정
D. 지갑 비밀번호 재설정
- 정답: B
- 해설: ch10은 가스를 비용이 아닌 실행 조건 모델로 정의한다.

### 문제 2
Gas Limit의 의미는?
A. 실제 소모량
B. 허용한 최대 실행 자원
C. 블록 크기
D. 계정 수
- 정답: B
- 해설: limit은 상한, used는 실제 소비량이다.

### 문제 3
Gas Used는 무엇을 뜻하는가?
A. 향후 예상 수수료
B. 실행에 실제로 소비된 가스
C. 주소 개수
D. 체인 ID
- 정답: B
- 해설: used는 실행 결과로 확정되는 소비량이다.

### 문제 4
Priority Fee의 목적은?
A. 키 복구
B. 우선 포함 유인 제공
C. 토큰 소각
D. 정책 공지
- 정답: B
- 해설: 혼잡 상황에서 포함 우선순위를 높이기 위한 추가 단가다.

### 문제 5
"가스는 수수료일 뿐"이 불완전한 이유는?
A. 수수료가 항상 무료라서
B. 실행 가능성과 실패 확률을 직접 좌우하기 때문
C. 주소가 없어서
D. 컨트랙트가 없어서
- 정답: B
- 해설: 가스는 비용을 넘어 실행 조건의 핵심 변수다.
