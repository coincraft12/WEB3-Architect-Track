# ch6.5. 블록체인 최소 정의 - Quiz

## 객관식 퀴즈 (4지선다)

### 문제 1
블록체인 최소 정의에 가장 가까운 것은?
A. 코인 가격 예측 시스템
B. 공유 상태 머신
C. 중앙 결제 서버
D. 메시지 앱
- 정답: B
- 해설: ch6.5는 블록체인을 공유 상태 머신으로 정의한다.

### 문제 2
상태(State)의 설명으로 적절한 것은?
A. 사용자 감정 상태
B. 체인의 현재 사실 집합
C. 지갑 UI 색상
D. 프로젝트 로드맵
- 정답: B
- 해설: 잔액/소유/컨트랙트 변수 등이 상태를 구성한다.

### 문제 3
합의의 목적은?
A. 키 생성
B. 어떤 트랜잭션 순서를 채택할지 고정
C. 수험표 출력
D. 이벤트 삭제
- 정답: B
- 해설: 합의는 단일 역사 순서를 만들어 네트워크 일관성을 확보한다.

### 문제 4
"공유 장부" 표현이 부족한 이유로 맞는 것은?
A. 장부라는 단어가 영어라서
B. 상태 전이와 순서 고정 개념이 빠져서
C. 블록이 없어서
D. 주소가 없어서
- 정답: B
- 해설: 블록체인은 기록 저장만이 아니라 상태 변경 규칙 시스템이다.

### 문제 5
키/주소가 필연적인 이유는?
A. UI 디자인을 위해
B. 상태 변경 요청의 주체 식별/승인을 위해
C. 채굴 장비 판매를 위해
D. 수수료를 숨기기 위해
- 정답: B
- 해설: 누가 어떤 행동을 승인했는지 확인하려면 키/주소 체계가 필요하다.
