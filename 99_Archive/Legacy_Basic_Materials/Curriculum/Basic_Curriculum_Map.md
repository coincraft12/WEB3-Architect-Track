# Basic Curriculum Map (Lecture Note -> Regulation)

기준 문서: `00_Governance/Qualification_Management_and_Operation_Regulations.md`

## 1. Basic 범위 정의
- 분산 신뢰 구조 이해
- WEB3 기초 구조 이해: 지갑/주소/트랜잭션/가스/스마트컨트랙트/토큰 기본 개념

## 2. 강의 노트 매핑
- `ch1~ch4`: 중앙화 신뢰 구조, 단일 기록 구조, 권한 집중, 분산 신뢰 3층
- `ch5~ch8`: 키 기반 소유권, 트랜잭션 구조, 지갑, 주소
- `ch9~ch11`: 트랜잭션 결과 해석, 가스 조건, 스마트컨트랙트 계정
- `ch12~ch14`: 기본 시나리오 구조 해석, 위험요소 분류, 점검 체크리스트
- `ch6.5`: 블록체인 최소 정의(연결 강의)
- `01_Intro`: 과정 관점 정렬

## 3. 정합성 판단
- 위 노트 묶음은 규정의 Basic 검정기준 및 검정과목 범위를 충족한다.
- Basic 단계에서 Associate/Professional의 설계 책임 범위를 요구하지 않는다.

## 4. 운영 메모
