# Module A. 지갑이라는 존재에 익숙해지기

## 1강. 지갑은 계정이 아니다

- 목표: 지갑에 대한 가장 위험한 오해 제거
- 핵심 메시지:
  - 지갑 ≠ 앱
  - 지갑 ≠ 계좌
  - 지갑 = 신원 + 권한 + 서명
- 통과 기준:
  - “지갑이 뭐냐”는 질문에 자기 말로 설명 가능

---

## 2강. 주소는 ID다

- 목표: 주소의 정체를 감각적으로 이해
- 다루는 내용:
  - 주소는 사람이다
  - 이름은 없고, 문자열만 있다
- 실습:
  - 내 주소 찾기
  - 주소 복사 / 공유
- 통과 기준:
  - 주소를 계정번호처럼 취급하지 않음

---

## 3강. 지갑 앱 ≠ 지갑

- 목표: 앱 의존 사고 제거
- 다루는 내용:
  - 같은 지갑을 다른 앱에서 복구하는 개념
  - 앱 삭제 ≠ 자산 삭제
- 시연:
  - “앱을 지워도 자산은 남아 있다”
- 통과 기준:
  - 지갑과 앱을 명확히 구분해서 설명 가능

---

## 4강. 지갑 안에 코인은 없다

- 목표: 소유 개념 바로잡기
- 다루는 내용:
  - 코인은 블록체인에 기록된다
  - 지갑은 ‘보여주는 창’이다
- 시각적 설명:
  - 잔고 화면 vs 트랜잭션 기록
- 통과 기준:
  - “지갑 안에 코인이 있다”는 표현을 쓰지 않음

---

# Module B. 생성과 백업 훈련

## 5강. 지갑 생성 실습

- 목표: 첫 진입 공포 제거
- 실습:
  - 신규 지갑 생성
  - 초기 화면 확인
- 강조:
  - 생성은 어렵지 않다
- 통과 기준:
  - 지갑 생성 완료

---

## 6강. 니모닉의 진짜 의미

- 목표: 복구 개념 정확히 인식
- 핵심 메시지:
  - 니모닉 = 복구 수단
  - 복구 가능 = 소유권
- 실습:
  - 니모닉 직접 확인
- 통과 기준:
  - 니모닉의 역할 설명 가능

---

## 7강. 백업 방식 선택 훈련

- 목표: 백업을 “선택”하게 만들기
- 다루는 방식:
  - 종이 백업
  - 금속 백업
  - 분산 보관 개념(이론)
- 토론 포인트:
  - 내 상황에 맞는 방식은?
- 통과 기준:
  - 자신의 백업 방식을 명확히 선택

---

## 8강. 잃어버렸을 때 시뮬레이션

- 목표: 책임 위치 고정
- 시나리오:
  - 니모닉 분실
  - 일부 유출
- 다루는 질문:
  - 고객센터가 왜 없는가
- 통과 기준:
  - “잃어버리면 끝”이라는 구조 인식

- 목표: 네트워크 개념 체득
- 핵심 메시지:
  - 네트워크는 세계다
- 실습:
  - 네트워크 변경
- 통과 기준:
  - 스스로 네트워크 전환 가능

---

## 11강. 코인 전송 실습

- 목표: 전송 공포 제거
- 실습:
  - 소액 테스트 코인 전송
- 다루는 것:

---

## 12강. 전송 후 확인 습관

- 목표: 보내고 끝내지 않기

  - 전송 후 반드시 결과 확인
