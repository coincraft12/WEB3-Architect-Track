# Associate Blueprint (Regulation-Aligned Draft)

기준 문서: `00_Governance/Qualification_Management_and_Operation_Regulations.md`

## 1. 목표 역량
- 서비스/프로젝트 구조를 WEB3 관점에서 해석
- 역할/권한/자산흐름 분석
- 구조적 취약점 및 운영 리스크 식별
- 개선 방향 제시

## 2. 검정 정합성
- 필기: WEB3 아키텍처 분석
- 실기: 권한 통제 설계 / 자금 흐름 기반 리스크 평가

## 3. 커리큘럼 초안 모듈
- Module A: 구조 해석 프레임 고도화 (주체/권한/행동/자산흐름)
- Module B: 권한 통제 설계 (승인, 서명, 역할 분리, 책임 경계)
- Module C: 자금 흐름 기반 리스크 식별 (입출금/브릿지/컨트랙트 상호작용)
- Module D: 시나리오 개선안 도출 (취약점 -> 통제안 -> 운영 체크)

## 4. 산출물 초안
- Architecture Read Sheet
- Authority Control Matrix
- Fund Flow Risk Sheet
- Scenario Improvement Note

## 5. 제한
- 응시자격: Basic 취득자
- 설계의 최종 책임 판단은 Professional 범위로 넘긴다.
